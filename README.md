# Sitio web sobre Ciberseguridad - CiberSegura

Este proyecto es un sitio web informativo sobre ciberseguridad. Forma parte de la evaluación del módulo y está desarrollado con **HTML5**, **CSS3**, **Bootstrap**, **JavaScript** y **jQuery**.
## Autor
Proyecto desarrollado como parte del módulo de portafolio del bootcamp Fullstack.

## Estructura del proyecto

- `index.html`: Página de inicio, presenta la introducción a la ciberseguridad.
- `amenazas.html`: Página con información sobre amenazas comunes (phishing, ransomware, malware, DDoS).
- `consejos.html`: Página con buenas prácticas, formulario de contacto y un modal con un test de seguridad.

Carpetas:

- `css/`
  - `style.css` → Estilos personalizados del sitio.
- `js/`
  - `main.js` → Lógica de interacción, validaciones y test.
- `img/`
  - Imágenes utilizadas en el sitio (por ejemplo, imagen principal de la portada).

## Tecnologías utilizadas

- HTML5 (etiquetas semánticas).
- CSS3.
- [Bootstrap 5](https://getbootstrap.com/).
- [jQuery](https://jquery.com/).
- Git para control de versiones.
- GitHub para el repositorio remoto.

## Funcionalidades principales

- Diseño responsivo utilizando Bootstrap.
- Tarjetas y acordeón para explicar amenazas comunes.
- Formulario de contacto con validación básica en JavaScript/jQuery.
- Modal con un test de seguridad que entrega retroalimentación al usuario.

## Cómo ejecutar el proyecto

1. Clonar o descargar este repositorio.
2. Abrir `index.html` en un navegador web.
3. Navegar entre las páginas mediante la barra de navegación.
